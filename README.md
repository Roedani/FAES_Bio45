# BIOF045: Next Generation Sequencing Data Analysis on the Cloud

| Audience | Computational skills required | Duration |
:----------|:-------------|:----------|
| Biologists | None | 1-day workshop |

### Description

This repository has teaching materials for a 1-day Introduction to shell workshop. This workshop focuses on teaching basic skills to use the command line interface, specifically `bash`, the lessons include `for` loops, shell scripting, positional parameters.

### Learning Objectives

1.	Understand the necessity for, and use of, the command line interface (bash/shell).

### Installation Requirements
***Mac users:***
X2Go for remote GUI
***Windows users:***
PuTTY and X2Go
